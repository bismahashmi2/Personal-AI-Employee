Hackathon Project Python Claude AI OAuth 2.0

What is Digital FTE?

Digital FTE (Full Time Equivalent) is an autonomous AI employee that monitors communications, processes tasks with human-in-the-loop approval, and manages business operations 24/7. Built with Claude Code for the AI Employee Hackathon.

View on GitHub My Portfolio

Key Features

📧 Gmail Integration

Monitor and process emails automatically via Gmail API

💼 LinkedIn Detection

Track trending topics and business opportunities

💬 WhatsApp Monitoring

Real-time message monitoring via Playwright

🤖 Claude AI Reasoning

Automatic plan generation and decision making

✅ Human-in-the-Loop

Secure approval workflow for sensitive actions

📊 Analytics Dashboard

Real-time KPIs in Obsidian Markdown

Tech Stack

  • Language: Python 3.12+
  • AI Engine: Claude Code (Anthropic API)
  • APIs: LinkedIn OAuth 2.0, Gmail API
  • Automation: Playwright, MCP servers
  • Storage: File-based messaging (Obsidian vault structure)
